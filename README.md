# ReactTelas

Proyecto inicial de React con Vite (JavaScript).

Pasos para ejecutar localmente:

1. Instala Node.js (incluye `npm`) desde https://nodejs.org/
2. En la carpeta del proyecto, instala dependencias:

```powershell
npm install
```

3. Arranca el servidor de desarrollo:

```powershell
npm run dev
```

4. Abre el navegador en la URL que muestre la terminal (normalmente `http://localhost:5173`).
